# ✊📄✂️ Rock Paper Scissors Game

A fun and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript** with a modern UI.

---

## 🚀 Features

* 🎮 Play against computer
* 📊 Live score tracking
* ✨ Smooth animations & hover effects
* 🎨 Modern and responsive design
* ⚡ Instant result display

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
rock-paper-scissor/
│── index.html
│── style.css
│── script.js
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start playing 😎
